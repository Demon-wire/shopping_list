      Shopping List App                                                                                           
                                                                                                                  
      A simple shopping list web application built with Django (Python) and HTML templates.                       
    
      This project provides a basic foundation for managing a personal shopping list. The app uses an SQLite
      database and Django's built-in admin and template system.                                                   
                                                                                                                  
      Tech Stack
    
      - Backend: Python / Django
      - Frontend: HTML (Django Templates)
      - Database: SQLite
    
      Project Structure
    
      shopping_list/
      ├── Einkaufsliste/   # Main Django app
      ├── mylist/          # App / config directory
      ├── manage.py        # Django CLI utility
      └── db.sqlite3       # SQLite database
    
      Status
    
      This project is unfinished and open for anyone who wants to continue or build upon it. Basic knowledge of
      Django is required to contribute.
    
      Getting Started
    
      pip install django
      python manage.py migrate
      python manage.py runserver
